
## Описание

В проекте реализован класс `StrClass` для работы со строками и текстовыми файлами. Программа считывает строку из файла, выделяет отдельные слова, находит вещественные числа с фиксированной точкой и сохраняет результаты обработки в выходной файл.

## Возможности

* чтение текста из файла;
* выделение слов из строки;
* поиск вещественных чисел с фиксированной точкой;
* вывод результатов на экран;
* сохранение результатов в файл;
* конструктор копирования;
* перегрузка оператора присваивания;
* перегрузка оператора вывода `<<`.

## Файлы проекта

* `StrClass.h` — описание класса;
* `StrClass.cpp` — реализация методов класса;
* `main.cpp` — демонстрация работы программы.

## Входные данные

Программа получает имя входного файла через аргументы командной строки.

Пример:

```bash
main input.txt
```

## Выходные данные

* список слов из исходной строки;
* список вещественных чисел с фиксированной точкой;
* файл `decision.txt` с результатами обработки.

## Запуск

```bash
g++ main.cpp StrClass.cpp -o main
./main input.txt
```
